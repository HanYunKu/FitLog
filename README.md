## Overview
FitLog is a full-stack application for tacking workouts and diet entries.

## Features
- Log daily workouts with exercise details
- Record dietary entries
- View and manage past workout and diet logs
- Full frontend–backend integration

## Notes
- This project was originally developed for a course assignment. Course-related files have been removed for public release.
